Less mixins

  • .u-right();
  • IE7 float: right drop bug fixes:
  • 1. If the float: right follows an element in the html structure that should be to its left (and not above it), then that preceding element(s) must be float: left. http://stackoverflow.com/questions/10981767/clean-css-fix-of-ie7s-float-right-drop-bug#answer-11437688
  • 2. Simply change the markup order so that the element floating right comes before the element to its left.

  • Also adds a .lt-ie8 class to hack inline-block for IE 7 and below.

Link mixins

  • .u-link__colors();
  • Pass this mixin no arguments and it will color your link states with the following defaults: :link (default state) pacific, :hover/:focus pacific-50, :visited teal, :active navy.
  • .u-link__colors(@c);
  • Pass this mixin one color it will be used on all of the following states of your link; :link (default state), :visited, :hover/:focus, :active.
  • :hover and :focus share the same colors throughout this mixin.
  • .u-link__colors(@c, @h);
  • Pass this mixin two colors it will use the first color for the :link and :visited states, and the second color for the :hover/:focus and :active states.
  • .u-link__colors(@c, @h, @v, @a);
  • Pass this mixin four colors to color :link, :hover/:focus, :visited, :active states respectively.
  • Note that this is basically the same as using the .u-link() mixin except that it allows you to stick with the same mixin naming convention for consistency.
  • .u-link(@c, @h, @v, @a);
  • This is the base mixin that all .u-link__colors() mixins use. Please refrain from using this mixin directly in order to promote a consistent use of mixin names fro coloring links throughout this project. Remember that if you need to set colors for all states of a link you should use .u-link__colors(@c, @h, @v, @a).
  • Remember that :hover and :focus share the same colors.
  • .u-link__border();
  • Forces the default bottom border on the :link and :hover states.
  • .u-link__no-border();
  • Turn off the default bottom border on the :link and :hover states.
  • .u-link__hover-border();
  • Turn off the default bottom border on the :link state and force a bottom border on the :hover state.
