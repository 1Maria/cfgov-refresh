Media query vars

Size calculations

  • @small-screens-min
  • @small-screens-max
  • @medium-screens-min
  • @medium-screens-max
  • @large-screens-min
  • @large-screens-max
  • @extralarge-screens-min
  • You shouldn't need to use these variables directly, they're used to create the @respond-to vars which you should use instead.

Interpolated media type and expressions

  • @respond-to-small-screens-min
  • @respond-to-small-screens-max
  • @respond-to-medium-screens-min
  • @respond-to-medium-screens-max
  • @respond-to-large-screens-min
  • @respond-to-large-screens-max
  • @respond-to-extralarge-screens-min
  • Use these variables to easily target screen size ranges.

Usage

  • @media @respond-to-medium-screens-min {
    ...
}

Compiles to:

@media only all and (min-width: 48em) {
    ...
}
